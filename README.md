# Astrophysics Simulator

An interactive astrophysics simulator built with React, TypeScript, Vite, and the HTML5 Canvas API. This project explores fundamental concepts in astronomy, orbital mechanics, gravity, 3D celestial rendering, and spacetime visualization through real-time simulations.

---

## Features

### Phase 1: 2D Rendering
- Interactive celestial body visualization
- Dynamic object creation
- Real-time simulation controls
- Zoom and camera movement

### Phase 2: Orbital Mechanics
- Newtonian gravity simulation
- Multi-body gravitational interactions
- Stable and unstable orbital systems
- Orbital trajectory visualization
- Adjustable simulation speed

### Phase 3: 3D Celestial Rendering
- 3D sphere projection
- Rotatable camera system
- Depth perception and scaling
- Realistic celestial body representation

### Phase 4: Spacetime Visualization
- Gravitational field visualization
- Spacetime curvature demonstrations
- Interactive physics exploration
- Educational relativity concepts

## Physics Concepts

The simulator demonstrates several important astrophysical principles:

### Universal Gravitation

Every object attracts every other object according to Newton's Law of Gravitation:

```
F = G(m₁m₂)/r²
```

Where:

- F = gravitational force
- G = gravitational constant
- m₁ = mass of object 1
- m₂ = mass of object 2
- r = distance between objects

### Orbital Motion

The simulator models:

- Circular orbits
- Elliptical orbits
- Escape trajectories
- Multi-body interactions

### Relativity Visualizations

Spacetime curvature demonstrations illustrate how massive objects influence the geometry of space.

## Installation

Clone the repository:

```bash
git clone https://github.com/G-Stylo/astrophysics-simulator.git
```

Navigate into the project directory:

```bash
cd astrophysics-simulator
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open your browser and navigate to:

```text
http://localhost:5173
```
